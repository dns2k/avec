# avec
Shell FFMPEG Encoder for Windows v1.3

this project aimed to create a simple ffmpeg encoder using windows explorer shell shortcut.

how to use:
1. download latest zeranoe ffmpeg for windows - static build (http://ffmpeg.zeranoe.com/builds/)
2. extract the ffmpeg.exe & ffplay.exe to c:\windows\avec
3. extract the icons.zip to c:\windows\avec\icons
4. double click on avec-v1.3.reg to install the avec shell extention to the windows explorer right click menu 
5. enjoy and start encoding... :)

nb: you can freely modify the ffmpeg parameter by editing the avec-v1.3.reg and 
    applying the changes you make by double click the avec-v1.3.reg again
